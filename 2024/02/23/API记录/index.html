API记录

生成二维码

  • 地址：https://api.oioweb.cn/api/qrcode/encode

  • 返回：JSON

  • 请求：GET

  • 示例：https://api.oioweb.cn/api/qrcode/encode?text=https://api.oioweb.cn&m=2&type=svg&size=15

参数 必填 类型 说明
text ✔ string 文本内容
m number 边距，可选值{0,10}，默认2
type string 二维码类型，可选值{"jpg","svg"}，默认SVG
size number 二维码大小，默认15，可选值5-30,svg类型下该参数无效

获取网站的favicon.ico

  • 地址：https://api.oioweb.cn/api/site/favicon
  • 返回：JSON
  • 请求：GET
  • 示例：https://api.oioweb.cn/api/site/favicon?url=https://api.oioweb.cn

返回必应每日一图的最近7张图

  • 地址：https://api.oioweb.cn/api/bing
  • 返回：JSON
  • 请求：GET
  • 示例：https://api.oioweb.cn/api/bing
返回参数 类型 说明
copyright string 图片版权信息
copyrightlink string 版权地址URL
title string 图片标题
url string 图片地址URL
